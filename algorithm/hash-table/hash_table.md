# 哈希函数和哈希表

## 哈希函数特征

1. 输入无穷的，输出是有限集
2. 相同的输入，得到相同输出
3. 不同的输入，得到相同输出（哈希碰撞）
4. 输出是均匀分布在有限集上面的
5. 在输出上对值取模，那么取模结果也是均匀分布的
